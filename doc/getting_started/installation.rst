============
Installation
============


Using latest github master source
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Clone latest version from github

.. code::

    $ git clone https://github.com/WUSTL-ORL/NeuroDOT_Py
    
    
Now go to the cloned folder and install manually 

 
.. code::

   $ cd NeuroDOT_py
   $ pip install -r requirements.txt
   $ pip install -e .
